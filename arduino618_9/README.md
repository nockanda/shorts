9.시리얼입력으로 스탭모터 2개의 목표위치가 서로 차이가 날 경우 똑같이 출발해서 똑같이 도착하도록 하려면 어떻게 해야겠는가?
   0입력 : 스탭모터1과 스탭모터 2가 동시에 0의 위치로 이동
   1입력 : 스탭모터1은 360도, 스탭모터2는 90도 회전하는데 동시에 도착할 것!
   2입력 : 스탭모터1은 270도, 스탭모터2는 -90도 회전하는데 동시에 도착할 것!
